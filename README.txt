Demo d'un Micro-Service qui permet de déployer des WS REST Controller basée sur Spring MVC.

URL pour tester les WS REST Controller :

	+ Liste des étudiants:		http://localhost:8080//students
	+ Affiche étudiant par id:	http://localhost:8080/students/id
	
	+ Liste des formations:		http://localhost:8080//students
	+ Affiche formation par id:	http://localhost:8080/students/id
